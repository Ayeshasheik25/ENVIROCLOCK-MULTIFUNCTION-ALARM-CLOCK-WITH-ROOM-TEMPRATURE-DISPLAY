# ENVIROCLOCK-MULTIFUNCTION-ALARM-CLOCK-WITH-ROOM-TEMPRATURE-DISPLAY
A multifunction alarm clock developed using the LPC2148 ARM7 microcontroller featuring RTC-based timekeeping,LM35 room temprature sensing through ADC,keypad input,external interrupt-driven menu control and buzzer alarm. Includes fully modular C code, LCD interfacing and hardware-tested implementation.
